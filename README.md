## 开始使用

这是一个用于 LED 选型的拓展，你可以在 **嘉立创EDA** 或 **EasyEDA** 专业版中使用，更加方便地查看 LED 的颜色。

1. 下载本拓展，在 **嘉立创EDA** 或 **EasyEDA** 专业版中安装

2. 在首页、原理图、PCB页面中，点击顶部菜单的 `LED助手`，即可打开本拓展
